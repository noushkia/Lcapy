Setting Conditions for Step Response¶

In [6]:
v = expr('u(t)')
V_i = v(s)
V_o = V_i * F
V_o.ZPK().expr
Out[6]:
$\displaystyle \frac{\frac{1}{C} \frac{1}{L}}{s \left(s + \frac{C R_{1} R_{2} + L}{2 C L R_{2}} - \frac{\sqrt{C^{2} R_{1}^{2} R_{2}^{2} - 2 C L R_{1} R_{2} - 4 C L R_{2}^{2} + L^{2}}}{2 C L R_{2}}\right) \left(s + \frac{C R_{1} R_{2} + L}{2 C L R_{2}} + \frac{\sqrt{C^{2} R_{1}^{2} R_{2}^{2} - 2 C L R_{1} R_{2} - 4 C L R_{2}^{2} + L^{2}}}{2 C L R_{2}}\right)}$

As you can see above, if the two terms under the square root operator become negative, we will have j(imaginary unit) in our response, this will cause the response to have some sine and cosine functions.
To prevent the response from having these functions, we must provide conditions where the two terms are not negative. In order to cover the mentioned condition we have: $$C^2R_1^2R_2^2-2CLR_1R_2-4CLR_2^2+L^2 >= 0$$ $$(CR_1R_2-L)^2 - 4CLR_2^2 >= 0$$ $$(CR_1R_2-L)^2 >= 4CLR_2^2$$ $$L-CR_1R_2 <= 2R_2\sqrt{CL} <= CR_1R_2-L$$
